# flask-oauth2-keycloak
Flask app with OAUTH2.0 and Keycloak

```bash
# Install poetry
pip install poetry

# Create a new project
poetry new flask-oauth2-flask
cd flask-oauth2-flask

# Add flask and flask-oauthlib
poetry add flask authlib

# Create a new file app.py
Update the app.py file with the code
  
flask run
```
